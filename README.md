# CodeGuessr

**CodeGuessr Arena** — это браузерная игра, в которой вам нужно как можно быстрее определить язык программирования по случайному фрагменту кода. Проект вдохновлён гео‑угадайками вроде GeoGuessr, но вместо улиц мира вы исследуете исходный код.

## Как играть

1. **Старт раунда** — нажмите «Новый раунд». Сервер выберет случайный файл из коллекции примеров и покажет небольшое окно кода.
2. **Навигация** — используйте стрелки ← ↑ → ↓ для перемещения по коду. Вы видите только небольшую область вокруг курсора («фонарик»), остальной код скрыт.
3. **Угадывание** — под кодом расположены кнопки с названиями языков. У вас есть **25 секунд** и **3 попытки**.
4. **Очки** — за правильный ответ начисляется `100 + оставшееся_время × 6` очков. Чем быстрее ответ, тем больше баллов.
5. **Лидерборд** — после регистрации ваши результаты сохраняются в таблице лидеров по каждому языку.

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/yourname/CodeGuessr.git
cd CodeGuessr
```

### 2. Установка зависимостей
Перейдите в папку сервера и установите пакеты:
```bash
cd my-server
npm install
```

### 3. Подготовка примеров кода
Сервер ожидает, что в папке `examples_dump` (на одном уровне с `my-server`) будут лежать файлы с исходным кодом, организованные по языкам.  
Если вы клонировали репозиторий, эта папка уже содержит значительную коллекцию примеров.  
**Важно**: минимальная длина файла — **50 строк**, иначе он не попадёт в игру.

### 4. Запуск сервера
```bash
npm start
```
Сервер запустится на порту `3000`. Вы увидите сообщение:
```
Сервер запущен на http://localhost:3000
Файлов для игры найдено: XXX
```

### 5. Игра
Откройте браузер и перейдите по адресу:  
[http://localhost:3000](http://localhost:3000)

Для входа в локальной сети используйте IP‑адрес вашего компьютера (будет показан в консоли).

## Авторизация

- **Регистрация**: логин и пароль
- **Вход** после регистрации.

## Лидерборд

После каждого успешного раунда результат отправляется на сервер. Для каждого языка хранятся 10 лучших попыток пользователя. Глобальная таблица показывает топ‑50 по каждому языку среди всех игроков.
